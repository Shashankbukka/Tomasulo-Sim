The simulator can be accessed through the index.html. 
Press the Load button to browse and choose the file. Two sample input files input1 and input2 have already been provided. 
New files can be in the .txt format and have to utilize the same code semantics.
Use the runOnestep or skip multiple steps by the buttons provided in the top left corner.

The main code is written in the tSimulator.js
